A clean, minimal personal portfolio built with React and Vite.

## Features

- Dark / Light mode toggle
- Sections: About, Experience, Education, Projects, Tech Stack, Socials
- Smooth animations with Framer Motion

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on **Vercel** with custom domain `bysudipx.online` via Hostinger DNS.

Build command:
```bash
npm install && npm run build
```

Output directory: `dist`
